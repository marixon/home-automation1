This project is aimed to develope a webapplication for home automation. The purpose is to build an application which detects the home automation assets through a local network scan, allows for their configuration and leverage their capabilities in the browser. Key example assets are: hik connect gate control system, IP cameras, tuya temperature and humidity sensors, tuya light switches. 

The product roadmap is:

1. CLI module to scan IP devices and recognize them by type and list the detected devices.
2. CLI module to connect the the recognized assets to retrieve, edit and store their configuration 
3. very simple web application to show the detected devices on a canvas

The project shall use python and create its own virtual environment using venv.
The project shall use repository https://github.com/marixon/home-automation1
